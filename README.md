# n8n-Analyze-corporate-action-impact-from-NSE-RSS-with-Google-Sheets-Gemini-and-Gmail
AI-powered Corporate Actions Impact Analyzer built with n8n, NSE RSS, Google Sheets, Gemini AI, and Gmail. Monitors corporate actions, matches portfolio holdings, calculates dividend and bonus impacts, sends automated email summaries, and logs every run for portfolio monitoring and compliance.
